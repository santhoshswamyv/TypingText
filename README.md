# *Typing Text*

## Features

- Generates a paragraph using the free [Metaphorpsum API](https://metaphorpsum.com/)  
- Calculates the time taken to type the paragraph (in seconds)  
- Counts only **actual typing mistakes**, ignoring repeated errors  
- Simple and user-friendly interface  
- Built with **HTML, CSS, and JavaScript**, utilizing the free API  

---

## Developed Using

**IDE :** Visual Studio Code  

**Languages :** HTML, CSS, JavaScript  

**API :** [Metaphorpsum](https://metaphorpsum.com/)  

---

## How to Run the Project

*Clone the project*

```bash
git clone https://github.com/santhoshswamyv/TypingText
```

*Go to the project directory*

```bash
cd TypingText
```

*Open `index.html` in your browser or host the project locally*
